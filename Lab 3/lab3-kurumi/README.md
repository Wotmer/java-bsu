## Описание проделанной работы

1. **Backend:** Java 17 + Spring Boot (REST API, JPA).
2. **Database:** PostgreSQL 15.
3. **Frontend:** Nginx + HTML/JS + CSS (адаптивный дизайн, анимации).
4. **Инфраструктура:** Docker Compose объединяет 3 контейнера в одну сеть.
5. **Функционал:** Кликер с сохранением прогресса в БД, механика смены фаз (картинок) каждые 100 кликов, система "
   Награды" после 200 кликов, кнопка сброса.

**Безопасность (Бонус):** Реализована защита CORS. Основной фронтенд (порт 80) имеет доступ к API, шпионский сайт (порт
81) получает отказ в доступе (403 Forbidden).
